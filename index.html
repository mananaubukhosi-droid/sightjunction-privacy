1. Introduction

Sightjunction (“we”, “our”, “us”) is a location-based social networking application. We are committed to protecting your personal information in accordance with the Protection of Personal Information Act (POPIA).

2. Information We Collect

2.1 Personal Information

  • Username
  • Profile information (bio, interests, pursuits)
  • Email address (if provided)

2.2 Location Data

  • Exact GPS location
  • Real-time location (while app is open, optional background mode)
  • Proximity radius (0–1000 meters)

Important: Exact distance between users is visible inside the app.

2.3 Device & Usage Data

  • Device type and operating system
  • App activity (profile views, scans, connections)
  • IP address

3. How We Use Your Information

  • To show nearby users based on location
  • To enable connections between users
  • To display distance and proximity results
  • To improve app performance
  • To prevent abuse and enforce safety rules

4. User Visibility & Social Features

  • Users can see nearby users by default
  • Visible profile data includes:
    • Username
    • Interests / pursuits
    • Approximate distance (0–1000m)
  • Users can control visibility using a toggle in settings

5. Location Tracking

  • Location is used to calculate proximity between users
  • App may run in background if user enables it
  • Exact GPS location is collected for proximity accuracy
  • Users can disable location access through device settings

6. Chat & Interactions

  • Users can send connection requests and chat after connecting
  • Messages are stored in our database (Firebase)
  • Users can block and report other users

7. Data Storage & Security

  • Data is stored using Firebase database services
  • We do not currently use end-to-end encryption
  • Access to user data is restricted to necessary systems only

Note: While we take reasonable steps to protect data, no system is completely secure.

8. Data Sharing

We do not sell personal data.

Data may be shared in the following cases:

  • With other users (based on visibility settings)
  • With Firebase (data hosting and storage provider)
  • If required by law or legal authorities

9. Data Retention & Deletion

  • Users can delete their account instantly inside the app
  • When deleted, user data is removed from active systems
  • Some data may remain in backups for a limited time

10. Children’s Privacy

Sightjunction is intended for users aged 16 and older. Users aged 16–17 require parental or guardian consent.

11. Your Rights (POPIA)

  • Access your personal data
  • Correct your data
  • Delete your account and data
  • Withdraw consent at any time

To exercise rights, contact: sightjunction.contact@gmail.com

12. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted within the app or on our GitHub page.
